Tutorial Penggunaan

Ikuti langkah berikut untuk menghasilkan file proxies.yaml dari template Excel.
  1. Centang Mode yang diinginkan pada bagian Choose Modes.
  2. Siapkan file Excel sesuai contoh: Download Template.
  3. Klik Upload Excel → pilih file Anda.
  4. Tekan tombol Parse Excel. Tabel Detected HOST akan terisi dan daftar BUG muncul bila ada.
  5. Klik Generate YAML untuk membuat konfigurasi berdasarkan host, bug, dan mode terpilih.
  6. Tekan Download YAML untuk menyimpan sebagai proxies.yaml.
  7. (Opsional) Unduh QR Saweria dengan tombol Download QR bila ingin berdonasi.
Tips: jika tombol Parse Excel tidak aktif, pastikan file sudah terpilih. Jika Generate YAML tidak aktif, pastikan minimal satu Mode dicentang.
